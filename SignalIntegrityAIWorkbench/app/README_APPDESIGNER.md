# App Designer note

`SignalIntegrityAIWorkbenchApp.m` is a **class-based `uifigure` application** equivalent to what MathWorks App Designer exports as code. It is intentionally kept as a single reviewable `.m` file under Git.

To round-trip with App Designer in newer releases, use **File → Import** and follow the dialog to wrap existing code, or paste layout sections into a new `.mlapp` if you prefer visual editing.
